측정 잡음 \mathbf{v}_k

측정 잡음 \mathbf{v}_k는 시스템에서 측정된 데이터에 포함된 불확실성을 모델링하기 위해 사용된다. 이 잡음은 일반적으로 가우시안 분포를 따르며, 평균이 0이고, 특정 공분산 행렬을 가지는 난수로 생성된다.

가우시안 잡음 생성

측정 잡음 \mathbf{v}_k를 생성하는 방법은 다음과 같다:

  1. 가우시안 분포: 측정 잡음은 평균이 0이고, 공분산 행렬이 \mathbf{R}_k인 다변량 가우시안 분포를 따른다.
\mathbf{v}_k \sim \mathcal{N}(0, \mathbf{R}_k)

여기서 \mathcal{N}(0, \mathbf{R}_k)는 평균이 0이고, 공분산 행렬이 \mathbf{R}_k인 다변량 가우시안 분포를 나타낸다.

  1. 공분산 행렬 \mathbf{R}_k: \mathbf{R}_k는 잡음의 강도를 나타내며, 각 측정값에 대한 불확실성을 정의한다. 이 행렬은 보통 다음과 같은 형태를 갖는다:
\mathbf{R}_k = \begin{bmatrix} \sigma_x^2 & 0 \\ 0 & \sigma_y^2 \end{bmatrix}

여기서 \sigma_x^2와 \sigma_y^2는 각각 x와 y 위치 측정값의 분산(즉, 잡음의 강도)을 나타낸다. 이 값들은 실험적 데이터나 시스템의 특성을 기반으로 설정된다.

  1. 랜덤 변수 생성: 실제 구현에서는 컴퓨터에서 사용할 수 있는 난수 생성기를 이용해 \mathbf{R}_k의 공분산 행렬에 기반한 가우시안 분포를 따르는 무작위 값을 생성한다.

예를 들어, Python의 numpy 라이브러리에서는 numpy.random.multivariate_normal(mean, cov) 함수를 사용해 다변량 가우시안 분포를 따르는 난수를 생성할 수 있다. 여기서 mean은 평균 벡터이고, cov는 공분산 행렬이다.

```python import numpy as np

# 평균은 0, 공분산 행렬 R_k 설정 mean = [0, 0] R_k = [[sigma_x2, 0], [0, sigma_y2]]

# 가우시안 잡음 생성 v_k = np.random.multivariate_normal(mean, R_k) ```

측정 잡음 \mathbf{v}_k는 다변량 가우시안 분포에서 추출된 무작위 변수로 생성된다. 이 잡음은 측정 장치나 환경에서 발생할 수 있는 불확실성을 모델링하기 위해 사용되며, 공분산 행렬 \mathbf{R}_k는 잡음의 특성을 결정한다. 측정값에 잡음을 추가함으로써 현실적인 측정 데이터가 반영된 시스템 상태 추정을 가능하게 한다.
