최적화 이론은 수학과 공학에서 중요한 분야로, 주어진 제약 조건 하에서 가능한 모든 해 중에서 최적의 해를 찾는 방법을 연구한다. 이는 자원을 효율적으로 배분하거나 시스템의 성능을 최대화 또는 최소화하는 데 사용된다.

선형 계획법 (Linear Programming)

선형 방정식과 부등식으로 구성된 제약 조건 하에서 선형 목적 함수를 최적화하는 방법이다.

이차 계획법 (Quadratic Programming)

목적 함수가 이차 함수이며 제약 조건은 선형인 최적화 문제를 다룬다.

비선형 최적화 (Nonlinear Optimization)

비선형 목적 함수나 제약 조건을 가진 최적화 문제를 해결하는 방법이다.

볼록 최적화 (Convex Optimization)

목적 함수와 제약 조건이 볼록인 경우에 전역 최적해를 효율적으로 찾는 최적화 방법이다.

정수 계획법 (Integer Programming)

변수들이 정수 값을 가져야 하는 최적화 문제로, 조합론적 특성을 갖는다.

조합 최적화 (Combinatorial Optimization)

이산적인 조합 중에서 최적의 해를 찾는 최적화 분야로, 그래프 이론과 밀접한 관련이 있다.

다목적 최적화 (Multi-objective Optimization)

여러 개의 상충되는 목적 함수를 동시에 최적화하여 균형 있는 해를 찾는 방법이다.

동적 계획법 (Dynamic Programming)

복잡한 문제를 더 작은 부분 문제로 분할하고 그 해를 결합하여 전체 문제를 해결하는 방법이다.

전역 최적화 (Global Optimization)

여러 국소 최적해 중에서 가장 우수한 전역 최적해를 찾는 것을 목표로 한다.

확률적 최적화 (Stochastic Optimization)

불확실성과 확률 요소를 고려하여 최적화를 수행하는 방법이다.

메타휴리스틱 알고리즘 (Metaheuristic Algorithms)

유전 알고리즘(Genetic Algorithm), 입자 군집 최적화(Particle Swarm Optimization) 등 자연 현상을 모방한 최적화 기법이다.

로버스트 최적화 (Robust Optimization)

데이터의 불확실성을 고려하여 최악의 상황에서도 성능이 보장되는 해를 찾는 방법이다.

세미데피니트 프로그래밍 (Semidefinite Programming)

반양정 정의성(Positive Semidefinite)을 가진 행렬을 이용하여 최적화하는 기법이다.

경사하강법 (Gradient Descent)

목적 함수의 기울기를 따라 가장 낮은 값을 찾는 방식으로, 기계 학습에서 많이 사용된다.

라그랑주 최적화 (Lagrangian Optimization)

라그랑주 승수(Lagrange Multipliers)를 사용하여 제약 조건이 있는 최적화 문제를 해결한다.

카루시-쿤-터커 조건 (Karush-Kuhn-Tucker Conditions)

비선형 제약 최적화 문제의 최적성 조건을 제공하는 이론이다.

조정 최적화 (Constrained Optimization)

제약 조건 하에서 목적 함수를 최적화하는 일반적인 방법론이다.

분산 최적화 (Distributed Optimization)

여러 개의 노드나 에이전트가 분산된 환경에서 협력하여 최적화를 수행하는 방법이다.

온라인 최적화 (Online Optimization)

실시간으로 도착하는 데이터를 기반으로 즉각적인 최적의 결정을 내리는 방법이다.

강화 학습 (Reinforcement Learning)

에이전트가 환경과 상호 작용하며 보상을 최대화하도록 정책을 학습하는 기계 학습의 한 분야이다.

확률적 경사하강법 (Stochastic Gradient Descent)

전체 데이터셋 대신 무작위로 선택된 일부 데이터를 사용하여 경사하강법을 수행하는 알고리즘이다.

함수 해석학적 최적화 (Functional Analysis Optimization)

무한 차원의 공간에서 함수의 최적화를 다루는 이론이다.

균형 최적화 (Equilibrium Optimization)

게임 이론에서 각 참가자의 전략이 서로 최적화를 이루는 상태를 분석한다.

지수적 최적화 (Exponential Optimization)

목적 함수나 제약 조건에 지수적 요소가 포함된 최적화 문제를 다룬다.

내삽 최적화 (Interior Point Methods)

최적화 문제를 해결하기 위해 feasible region 내부를 따라가는 알고리즘이다.

부분평균법 (Subgradient Methods)

비미분 가능 함수의 최적화를 위한 방법으로, 미분 대신 부분 기울기를 사용한다.

병렬 최적화 (Parallel Optimization)

컴퓨팅 자원을 활용하여 최적화 알고리즘을 병렬로 실행함으로써 속도를 향상시킨다.

모델 예측 제어 (Model Predictive Control)

동적 시스템에서 미래의 예측 정보를 사용하여 현재의 최적 제어 입력을 계산한다.

순차적 이차 계획법 (Sequential Quadratic Programming)

비선형 최적화 문제를 일련의 이차 계획 문제로 근사하여 푸는 방법이다.

변분법 (Calculus of Variations)

함수의 함수인 범함수의 극값을 찾는 수학적 방법으로, 물리학과 공학에서 많이 활용된다.
