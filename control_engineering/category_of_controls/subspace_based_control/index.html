서브스페이스 기반 제어는 시스템의 상태 공간 모델을 모를 때, 입력과 출력 데이터를 기반으로 서브스페이스 식별 기법을 사용하여 제어기를 설계하는 방법이다. 이는 다변량 시스템에서 입력과 출력 간의 관계를 효과적으로 분석하고, 시스템의 고유 특성을 반영하는 서브스페이스를 추출하여 제어에 활용한다. 모델 식별 없이도 성능을 유지할 수 있어, 데이터가 충분히 확보된 시스템에서 많이 사용된다.
