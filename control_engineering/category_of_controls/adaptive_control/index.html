적응 제어(Adaptive Control)는 시스템의 동작이나 환경이 변할 때, 성능을 유지하기 위해 제어기의 파라미터를 자동으로 조정하는 제어 기법이다. 일반적인 제어 시스템은 미리 정의된 모델을 기반으로 하지만, 적응 제어는 모델의 불확실성이나 시스템 동작의 변화에 대응할 수 있다. 특히, 시스템의 동적 특성이 시간에 따라 달라지거나 외부 환경이 변화할 때, 이를 보정하여 안정성과 성능을 유지할 수 있다. 적응 제어는 항공, 로봇, 공정 제어 등 다양한 산업에서 사용되며, 시스템의 복잡성을 줄이고 실시간으로 대응할 수 있는 장점이 있다.

  • 모델 참조 적응 제어(Model Reference Adaptive Control, MRAC)
  • 자율 적응 제어(Self-tuning Control, STC)
  • 슬라이딩 모드 적응 제어(Sliding Mode Adaptive Control, SMAC)
