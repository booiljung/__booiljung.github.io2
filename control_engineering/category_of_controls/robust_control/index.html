강건 제어는 시스템에 불확실성이 존재하더라도 시스템의 안정성과 성능을 보장하는 제어 기법이다. 이는 모델링의 불완전성, 외란, 파라미터 변화에 강인한 제어기를 설계하는 데 초점을 맞추고 있다. 즉, 시스템의 모델이 정확하지 않거나 외부에서 예측할 수 없는 간섭이 발생하더라도 원하는 제어 성능을 유지하도록 한다.

대표적인 강건 제어 기법으로는 H_{\infty} 제어와 슬라이딩 모드 제어(Sliding Mode Control)가 있다. 이들은 시스템의 불확실성이나 외부 간섭에 대해 강건하게 설계되었으며, 주로 항공기 제어, 로봇 공학, 자동차 제어 등에 널리 활용된다.

강건 제어는 시스템이 모델링의 불확실성이나 외란에도 불구하고 안정성과 성능을 유지할 수 있도록 설계된 제어 기법이다.

  • H∞ 제어 (H-infinity Control)
  • μ-합성 (Mu-Synthesis)
  • 슬라이딩 모드 제어 (Sliding Mode Control)
  • LQG/LTR (Linear Quadratic Gaussian with Loop Transfer Recovery)
