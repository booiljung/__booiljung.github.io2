PI 제어는 비례 제어(비례 이득)와 적분 제어(적분 이득)를 결합한 제어 기법으로, 시스템의 오차를 줄이고 목표 상태로 빠르게 수렴하게 하는 데 사용된다. 비례 제어는 현재 오차 값에 비례하여 조치를 취하지만, 오차가 지속될 경우 안정적인 상태에 도달하지 못하고 작은 오차가 남을 수 있다. 이러한 오차를 제거하기 위해 적분 제어가 추가된다.

적분 제어는 시간이 지남에 따라 오차를 누적하여 작은 오차도 점차적으로 줄여 나간다. 적분 제어를 통해 시스템이 목표값에 더 정확히 도달할 수 있게 되지만, 잘못 설계된 적분 제어는 오버슈트나 진동을 유발할 수 있다. 따라서 비례 이득과 적분 이득의 적절한 조정이 필요하다.

모델 참조 적응 제어(Model Reference Adaptive Control, MRAC) 기법은 제어 시스템이 동적 특성을 갖는 환경에서 목표 성능을 유지하기 위해 제어기를 실시간으로 조정하는 방식이다. 아래에 MRAC에 속하는 주요 제어 기법들을 나열하고 각각에 대해 설명하겠다.
