행렬 덧셈의 교환 법칙

행렬 덧셈의 교환 법칙은 두 행렬을 덧셈할 때 순서에 상관없이 결과가 동일하다는 것을 의미한다. 즉, 행렬 \mathbf{A}와 \mathbf{B}에 대해 다음이 성립한다:

\mathbf{A} + \mathbf{B} = \mathbf{B} + \mathbf{A}

여기서 \mathbf{A}와 \mathbf{B}는 같은 크기를 가져야 한다.

행렬 덧셈의 결합 법칙

행렬 덧셈의 결합 법칙은 세 행렬을 덧셈할 때 덧셈 순서에 상관없이 결과가 동일하다는 것을 의미한다. 즉, 행렬 \mathbf{A}, \mathbf{B} 및 \mathbf{C}에 대해 다음이 성립한다:

(\mathbf{A} + \mathbf{B}) + \mathbf{C} = \mathbf{A} + (\mathbf{B} + \mathbf{C})

여기서 \mathbf{A}, \mathbf{B} 및 \mathbf{C}는 같은 크기를 가져야 한다.

영행렬의 존재

모든 행렬에 대해 덧셈 항등원으로 작용하는 영행렬 \mathbf{0}가 존재한다. 즉, 행렬 \mathbf{A}에 대해 다음이 성립한다:

\mathbf{A} + \mathbf{0} = \mathbf{0} + \mathbf{A} = \mathbf{A}

여기서 영행렬 \mathbf{0}는 \mathbf{A}와 같은 크기를 가지는 모든 원소가 0인 행렬이다.

행렬의 덧셈에 대한 역행렬의 존재

각 행렬 \mathbf{A}에 대해 덧셈에 대한 역행렬 -\mathbf{A}가 존재한다. 이는 다음을 만족한다:

\mathbf{A} + (-\mathbf{A}) = (-\mathbf{A}) + \mathbf{A} = \mathbf{0}

여기서 -\mathbf{A}는 \mathbf{A}의 모든 원소를 음수로 변환한 행렬이다.

행렬 뺄셈의 분배 법칙

행렬의 덧셈 및 뺄셈은 다음의 법칙을 따른다:

\mathbf{A} - (\mathbf{B} + \mathbf{C}) = \mathbf{A} - \mathbf{B} - \mathbf{C}

여기서 \mathbf{A}, \mathbf{B} 및 \mathbf{C}는 같은 크기를 가져야 한다. 이는 행렬 뺄셈의 분배 법칙을 의미한다.

실수와의 상호작용

행렬 \mathbf{A}와 실수 c, d에 대해 다음이 성립한다:

(c+d)\mathbf{A} = c\mathbf{A} + d\mathbf{A}
c(\mathbf{A} + \mathbf{B}) = c\mathbf{A} + c\mathbf{B}

여기서 \mathbf{A}와 \mathbf{B}는 같은 크기를 가져야 한다.
