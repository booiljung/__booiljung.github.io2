1. 대각합과 대각행렬식

행렬 \mathbf{A}의 특성 방정식 \det(\mathbf{A} - \lambda \mathbf{I}) = 0의 계수는 행렬 \mathbf{A}의 고유값에 의해 영향을 받는다. 특히, 행렬 \mathbf{A}의 대각합은 고유값들의 합과 같고, 행렬식은 고유값들의 곱과 같다.

  • \mathbf{A} = [a_{ij}]이 n \times n $$ 행렬이라고 하면, 특성 방정식을 다음과 같이 전개할 수 있다.
\det(\mathbf{A} - \lambda \mathbf{I}) = (-1)^n \lambda^n + c_{n-1} \lambda^{n-1} + \cdots + c_1 \lambda + c_0 = 0

여기서 c_{n-1}는 \mathbf{A}의 대각합 (trace, \text{tr}(\mathbf{A}))의 음수와 동일하고, c_0는 행렬식 (determinant, \det(\mathbf{A}))과 같다.

\text{tr}(\mathbf{A}) = \sum_{i=1}^{n} \lambda_i
\det(\mathbf{A}) = \prod_{i=1}^{n} \lambda_i

여기서 \lambda_1, \lambda_2, \ldots, \lambda_n 이 \mathbf{A}의 고유값들이다.

2. 멱등행렬과 특성 방정식

멱등행렬 \mathbf{A}는 \mathbf{A}^k = \mathbf{A} (일반적으로 k = 2)을 만족시키는 행렬이다. 멱등행렬의 고유값은 0 또는 1이다.

  • 이는 \mathbf{A}가 멱등이라고 가정했을 때, 특성 방정식은 다음과 같이 간단해진다.
\det(\mathbf{A} - \lambda \mathbf{I}) = \lambda (\lambda - 1)^n = 0

따라서, 모든 고유값이 0이나 1이라는 것을 확인할 수 있다.

3. 행렬의 거듭제곱과 특성 방정식

행렬 \mathbf{A}의 거듭제곱 \mathbf{A}^k의 고유값은 \mathbf{A}의 고유값의 k제곱과 같다. 이는 특성 방정식의 근들을 각각 k제곱하는 것으로 해석할 수 있다.

  • 예를 들어, \mathbf{A}의 고유값이 \lambda_i인 경우, \mathbf{A}^k의 고유값은 \lambda_i^k이다. 이를 통해 특성 방정식 역시 변형된다.
\det(\mathbf{A}^k - \lambda \mathbf{I}) = \prod_{i=1}^{n} (\lambda_i^k - \lambda) = 0

4. 행렬의 대각화 가능성과 특성 방정식

행렬 \mathbf{A}가 대각화 가능하다면, \mathbf{A}는 다음과 같은 형태로 표현될 수 있다:

\mathbf{A} = \mathbf{P} \mathbf{\Lambda} \mathbf{P}^{-1}

여기서 \mathbf{P}는 \mathbf{A}의 고유벡터들로 구성된 행렬이고, \mathbf{\Lambda}는 \mathbf{A}의 고유값들로 구성된 대각행렬이다. 이렇게 대각화된 행렬의 특성 방정식은 다음과 같이 간단한다:

\det(\mathbf{\Lambda} - \lambda \mathbf{I}) = 0

이는 행렬 \mathbf{A}의 특성 방정식과 동일한 결과를 제공한다.

5. 비대각화 행렬과 특성 방정식

비대각화 행렬의 경우에도 특성 방정식은 정의될 수 있다. 예를 들어, 2 \times 2 행렬 \mathbf{A}가 다음과 같이 주어졌다고 하겠다:

\mathbf{A} = \begin{pmatrix} a & b \\ c & d \end{pmatrix}

이 경우, 특성 방정식은 다음과 같이 작성된다:

\det(\mathbf{A} - \lambda \mathbf{I}) = \begin{vmatrix} a - \lambda & b \\ c & d - \lambda \end{vmatrix} = (a - \lambda)(d - \lambda) - bc = 0

이로부터 고유값을 구할 수 있다:

\lambda^2 - (a + d)\lambda + (ad - bc) = 0

이와 같은 방정식을 해석하면 비대각화 행렬의 고유값들을 얻을 수 있다.

6. 단위 행렬과 특성 방정식

단위 행렬 \mathbf{I}는 모든 주대각선상 원소가 1이고 나머지 원소는 0인 정사각 행렬이다. 단위 행렬의 고유값은 모두 1이다. 따라서, n \times n 단위 행렬 \mathbf{I}의 특성 방정식은 매우 단순한다:

\det(\mathbf{I} - \lambda \mathbf{I}) = \det(\mathbf{I - \lambda I}) = \det((1 - \lambda) \mathbf{I}) = (1 - \lambda)^n = 0

7. 차원 축소와 특성 방정식

고차원 행렬에서 저차원 하위 행렬 (Principal Submatrix)로의 차원 축소가 이루어질 때, 그 하위 행렬의 특성 방정식은 원래 행렬의 특성 방정식의 일부 정보를 유지한다. 하위 행렬의 고유값은 원래 행렬의 고유값들의 부분 집합이 될 수 있다.

예를 들어, 3 \times 3 행렬 \mathbf{A}의 하위 행렬을 고려할 때, 그 하위 행렬에서 계산된 고유값은 \mathbf{A}의 특성 방정식에서 유도될 수 있다.

행렬의 특성 방정식은 행렬 분석에서 중요한 도구이다. 이를 통해 행렬의 고유값을 구하고, 행렬의 성질, 예를 들어 대각합, 행렬식 및 대각화 가능성 등의 정보를 얻을 수 있다. 멱등행렬, 단위행렬과 같은 특수한 행렬들에도 특성 방정식은 유용하게 사용된다.

이 모든 내용을 종합하면, 특성 방정식은 선형대수학에서 강력한 수학적 도구로, 다양한 차원의 행렬과 그 응용을 다루는 데 중요한 역할을 한다.
