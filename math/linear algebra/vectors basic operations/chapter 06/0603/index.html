벡터의 정규화는 주어진 벡터의 방향은 유지하면서 크기(길이)를 1로 만드는 과정이다. 이를 통해 우리는 단위 벡터를 얻을 수 있다.

벡터의 크기

임의의 벡터 \mathbf{v}가 주어졌다고 가정하겠다. 이 벡터는 여러 성분으로 이루어질 수 있다. 예를 들어, 2차원 벡터 \mathbf{v} = [v_x, v_y]와 3차원 벡터 \mathbf{v} = [v_x, v_y, v_z]가 있다.

벡터의 크기(또는 길이)는 각 성분의 제곱의 합의 제곱근으로 나타낼 수 있다. 수학적으로 벡터 \mathbf{v} = [v_1, v_2, ..., v_n]의 크기 \|\mathbf{v}\|는 다음과 같이 정의된다:

\|\mathbf{v}\| = \sqrt{v_1^2 + v_2^2 + ... + v_n^2}

벡터의 정규화 과정

벡터 \mathbf{v}를 정규화하는 과정은 \mathbf{v}의 각 성분을 \|\mathbf{v}\|로 나누는 것으로 이루어진다. 이를 통해 우리는 크기가 1인 벡터, 즉 단위 벡터 \mathbf{u}를 얻게 된다:

\mathbf{u} = \frac{\mathbf{v}}{\|\mathbf{v}\|}

좀 더 구체적으로, \mathbf{v} = [v_x, v_y] 일 경우 단위 벡터 \mathbf{u}는 다음과 같이 계산된다:

\mathbf{u} = \left[\frac{v_x}{\|\mathbf{v}\|}, \frac{v_y}{\|\mathbf{v}\|}\right]

3차원 벡터 \mathbf{v} = [v_x, v_y, v_z]의 경우 단위 벡터 \mathbf{u}는 다음과 같이 계산된다:

\mathbf{u} = \left[\frac{v_x}{\|\mathbf{v}\|}, \frac{v_y}{\|\mathbf{v}\|}, \frac{v_z}{\|\mathbf{v}\|}\right]

예제

다음은 벡터를 정규화하는 과정을 실제로 수행해보는 예제이다.

2차원 벡터 예제

주어진 벡터 \mathbf{v} = [3, 4]를 정규화해보겠다.

  1. 벡터 \mathbf{v}의 크기를 구한다:

$$ |\mathbf{v}| = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = 5

$$

  1. 각 성분을 벡터의 크기로 나눈다:

$$ \mathbf{u} = \left[\frac{3}{5}, \frac{4}{5}\right] = [0.6, 0.8]

$$

3차원 벡터 예제

주어진 벡터 \mathbf{v} = [1, 2, 2]를 정규화해보겠다.

  1. 벡터 \mathbf{v}의 크기를 구한다:

$$ |\mathbf{v}| = \sqrt{1^2 + 2^2 + 2^2} = \sqrt{1 + 4 + 4} = \sqrt{9} = 3

$$

  1. 각 성분을 벡터의 크기로 나눈다:

$$ \mathbf{u} = \left[\frac{1}{3}, \frac{2}{3}, \frac{2}{3}\right]

$$

벡터 정규화의 응용

벡터의 정규화 과정은 다양한 분야에서 매우 유용하게 사용된다. 특히 컴퓨터 그래픽스, 물리 시뮬레이션, 데이터 분석 등에서 자주 활용된다. 다음은 벡터 정규화의 주요 응용 예시들이다.

컴퓨터 그래픽스

컴퓨터 그래픽스에서 벡터 정규화는 조명 계산, 표면의 법선 벡터 계산 등 다양한 용도로 사용된다. 단위 벡터는 방향 정보를 제공하되 크기를 1로 유지하여 계산을 간단하게 만든다.

물리 시뮬레이션

물리 시뮬레이션에서는 속도 벡터, 가속도 벡터 등을 정규화하여 방향성을 유지하되 특정 크기를 부여하는 경우가 많다. 예를 들어, 물체의 이동 방향을 계산할 때 벡터를 정규화하여 방향을 설정하고 이후 특정 속도를 적용할 수 있다.

데이터 분석

데이터 분석에서는 벡터 정규화를 통해 데이터 포인트를 비교하기 쉽게 만들 수 있다. 특히 텍스트 마이닝에서 단어 빈도 벡터를 정규화하여 비교하거나, 머신 러닝에서 특성을 정규화하는 과정에서 사용된다.

벡터 정규화의 주의사항

벡터 정규화가 유용하지만 몇 가지 주의사항도 있다. 특히 벡터의 크기가 0인 경우 정규화할 수 없다는 점을 유의해야 한다.

벡터의 크기가 0인 경우

만약 벡터 \mathbf{v}의 크기가 0이라면, 그 벡터는 모든 성분이 0이라는 의미이다. 이 경우에는 벡터를 정규화할 수 없다. 수학적으로 \frac{0}{0}은 정의되지 않기 때문이다. 따라서 정규화 과정에서는 우선 벡터의 크기가 0인지 확인하는 것이 중요하다.

구현 예제 (Python 코드)

다음은 벡터를 정규화하는 파이썬 코드 예제이다.

import numpy as np

def normalize_vector(v):
    norm = np.linalg.norm(v)
    if norm == 0:
        raise ValueError("Zero vector cannot be normalized")
    return v / norm

v1 = np.array([3, 4])
v2 = np.array([1, 2, 2])

u1 = normalize_vector(v1)
u2 = normalize_vector(v2)

print("Original vector v1:", v1)
print("Normalized vector u1:", u1)
print("Original vector v2:", v2)
print("Normalized vector u2:", u2)

이 코드에서는 벡터를 인풋으로 받아 크기가 0인지 확인한 후 정규화하는 과정을 거친다. 그 결과로 단위 벡터를 반환한다.
