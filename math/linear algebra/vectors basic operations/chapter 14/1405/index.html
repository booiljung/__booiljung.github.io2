문제 1

벡터 \mathbf{a} = 3\mathbf{i} + 4\mathbf{j}의 방향을 구하시오.

풀이

벡터의 방향은 다음과 같이 구할 수 있다:

\theta = \tan^{-1} \left( \frac{a_y}{a_x} \right)

여기서 \mathbf{a} = 3\mathbf{i} + 4\mathbf{j}이므로, a_x = 3이고 a_y = 4이다.

문제 4

벡터 \mathbf{d}가 45^\circ의 방향을 갖고 크기가 10인 경우, 이 벡터의 성분을 구하시오.

풀이

벡터 \mathbf{d}의 크기와 방향이 주어졌을 때, 벡터의 성분을 구할 수 있다. 벡터 \mathbf{d}의 성분 d_x와 d_y는 다음과 같이 구할 수 있다:

d_x = d \cos \theta
d_y = d \sin \theta

여기서 \mathbf{d}의 크기 d = 10 그리고 \theta = 45^\circ 이다.

문제 6

벡터 \mathbf{f} = x\mathbf{i} + y\mathbf{j}과 벡터 \mathbf{g} = -y\mathbf{i} + x\mathbf{j}가 서로 수직임을 증명하시오.

풀이

벡터 \mathbf{f}와 \mathbf{g}가 수직이라는 것은 내적이 0임을 의미한다. 즉, 다음과 같아야 한다:

\mathbf{f} \cdot \mathbf{g} = (x\mathbf{i} + y\mathbf{j}) \cdot (-y\mathbf{i} + x\mathbf{j}) = 0

이를 계산하면:

\mathbf{f} \cdot \mathbf{g} = x(-y) + y(x) = -xy + xy = 0

따라서, \mathbf{f}와 \mathbf{g}는 수직이다.

문제 7

벡터 \mathbf{h}의 크기가 5이고, \mathbf{i}의 크기가 12이며, \mathbf{h}와 \mathbf{i}의 각도가 30^\circ일 때 두 벡터의 내적을 구하시오.

풀이

벡터 \mathbf{h}와 \mathbf{i}의 내적은 다음과 같이 구할 수 있다:

\mathbf{h} \cdot \mathbf{i} = |\mathbf{h}| |\mathbf{i}| \cos \theta

여기서, |\mathbf{h}| = 5, |\mathbf{i}| = 12, \theta = 30^\circ이다.

문제 8

벡터 \mathbf{j}와 \mathbf{k}의 합 $$\mathbf{j} + \mathbf{k}가 주어졌을 때, 두 벡터의 합의 크기를 구하시오. \mathbf{j} = 6\mathbf{i} + 8\mathbf{j}, \mathbf{k} = -3\mathbf{i} + 4\mathbf{j} 이다.

풀이

먼저 벡터 \mathbf{j} + \mathbf{k}를 구한다:

\mathbf{j} + \mathbf{k} = (6\mathbf{i} + 8\mathbf{j}) + (-3\mathbf{i} + 4\mathbf{j}) = (6 - 3)\mathbf{i} + (8 + 4)\mathbf{j} = 3\mathbf{i} + 12\mathbf{j}

그런 다음, 벡터 \mathbf{j} + \mathbf{k}의 크기를 구한다:

| \mathbf{j} + \mathbf{k} | = \sqrt{(3)^2 + (12)^2} = \sqrt{9 + 144} = \sqrt{153} = 3\sqrt{17}

문제 9

벡터 \mathbf{l} = 2\mathbf{i} - 3\mathbf{j}와 벡터 \mathbf{m} = 4\mathbf{i} + \mathbf{j}가 이루는 각도를 구하시오.

풀이

벡터 \mathbf{l}과 \mathbf{m}의 내적을 이용하면 각도를 구할 수 있다:

\mathbf{l} \cdot \mathbf{m} = (2\mathbf{i} - 3\mathbf{j}) \cdot (4\mathbf{i} + \mathbf{j}) = 2 * 4 + (-3) * 1 = 8 - 3 = 5

두 벡터의 크기는 각각:

| \mathbf{l} | = \sqrt{(2)^2 + (-3)^2} = \sqrt{4 + 9} = \sqrt{13}
| \mathbf{m} | = \sqrt{(4)^2 + (1)^2} = \sqrt{16 + 1} = \sqrt{17}

따라서, 각도 \theta는 다음과 같이 구할 수 있다:

\cos \theta = \frac{\mathbf{l} \cdot \mathbf{m}}{|\mathbf{l}| |\mathbf{m}|} = \frac{5}{\sqrt{13} \sqrt{17}}

이 값을 계산하여 \theta를 구한다:

\theta = \cos^{-1} \left( \frac{5}{\sqrt{13} \sqrt{17}} \right)
