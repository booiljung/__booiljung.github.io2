벡터 덧셈의 교환법칙

벡터의 덧셈에서 교환법칙(Commutative Property)이란, 두 벡터의 덧셈 순서를 바꿔도 결과가 동일하다는 것을 의미한다. 수학적으로 이는 다음과 같이 표현된다:

\mathbf{a} + \mathbf{b} = \mathbf{b} + \mathbf{a}

여기서 \mathbf{a}와 \mathbf{b}는 벡터이다.

예제

만약 \mathbf{a} = \begin{pmatrix}a_1 \\ a_2\end{pmatrix} 와 \mathbf{b} = \begin{pmatrix}b_1 \\ b_2\end{pmatrix} 라면,

\mathbf{a} + \mathbf{b} = \begin{pmatrix}a_1 \\ a_2\end{pmatrix} + \begin{pmatrix}b_1 \\ b_2\end{pmatrix} = \begin{pmatrix}a_1 + b_1 \\ a_2 + b_2\end{pmatrix}

반면에,

\mathbf{b} + \mathbf{a} = \begin{pmatrix}b_1 \\ b_2\end{pmatrix} + \begin{pmatrix}a_1 \\ a_2\end{pmatrix} = \begin{pmatrix}b_1 + a_1 \\ b_2 + a_2\end{pmatrix}

여기서 \mathbf{a} + \mathbf{b} = \mathbf{b} + \mathbf{a} 이므로, 벡터 덧셈의 교환법칙이 성립한다는 것을 알 수 있다.

벡터 덧셈의 결합법칙

벡터의 덧셈에서 결합법칙(Associative Property)이란, 세 벡터를 덧셈할 때 덧셈의 순서에 상관없이 결과가 동일하다는 것을 의미한다. 수학적으로 이는 다음과 같이 표현된다:

(\mathbf{a} + \mathbf{b}) + \mathbf{c} = \mathbf{a} + (\mathbf{b} + \mathbf{c})

여기서 \mathbf{a}, \mathbf{b}, \mathbf{c}는 벡터이다.

예제

만약 \mathbf{a} = \begin{pmatrix}a_1 \\ a_2\end{pmatrix}, \mathbf{b} = \begin{pmatrix}b_1 \\ b_2\end{pmatrix}, \mathbf{c} = \begin{pmatrix}c_1 \\ c_2\end{pmatrix}라면,

(\mathbf{a} + \mathbf{b}) + \mathbf{c} = \begin{pmatrix}a_1 + b_1 \\ a_2 + b_2\end{pmatrix} + \begin{pmatrix}c_1 \\ c_2\end{pmatrix} = \begin{pmatrix}(a_1 + b_1) + c_1 \\ (a_2 + b_2) + c_2\end{pmatrix}

반면에,

\mathbf{a} + (\mathbf{b} + \mathbf{c}) = \begin{pmatrix}a_1 \\ a_2\end{pmatrix} + \begin{pmatrix}b_1 + c_1 \\ b_2 + c_2\end{pmatrix} = \begin{pmatrix}a_1 + (b_1 + c_1) \\ a_2 + (b_2 + c_2)\end{pmatrix}

여기서 (\mathbf{a} + \mathbf{b}) + \mathbf{c} = \mathbf{a} + (\mathbf{b} + \mathbf{c}) 이므로, 벡터 덧셈의 결합법칙이 성립한다는 것을 알 수 있다.
