선형 결합

선형 결합(linear combination)은 벡터를 합치거나 조합하여 새로운 벡터를 만드는 과정이다. 두 개 이상의 벡터와 이 벡터들에 스칼라 값을 곱해 더하여 만들어진 벡터로 정의된다.

즉, 벡터 \mathbf{v}_1, \mathbf{v}_2, \cdots, \mathbf{v}_n와 스칼라 a_1, a_2, \cdots, a_n가 주어졌을 때, 벡터 \mathbf{v}는 다음과 같은 형태로 표현된다.

\mathbf{v} = a_1 \mathbf{v}_1 + a_2 \mathbf{v}_2 + \cdots + a_n \mathbf{v}_n

위의 식처럼, 스칼라 변수들 a_1, a_2, \cdots, a_n와 벡터들 \mathbf{v}_1, \mathbf{v}_2, \cdots, \mathbf{v}_n 모두 주어졌을 때, 이는 벡터 \mathbf{v}를 나타내는 하나의 값이다.

벡터 공간

벡터 공간(vector space)은 벡터의 집합으로서, 이러한 벡터들 사이에서 덧셈, 스칼라 곱 등이 정의되며, 이들 연산이 다음과 같은 공리(axiom)를 만족하는 구조를 가진 집합이다.

벡터 공간의 공리

  1. 벡터 덧셈의 닫힘성: 임의의 두 벡터 \mathbf{u}, \mathbf{v} \in V에 대해, \mathbf{u} + \mathbf{v} \in V.
  2. 벡터 덧셈의 교환법칙: 임의의 두 벡터 \mathbf{u}, \mathbf{v} \in V에 대해, \mathbf{u} + \mathbf{v} = \mathbf{v} + \mathbf{u}.
  3. 벡터 덧셈의 결합법칙: 임의의 세 벡터 \mathbf{u}, \mathbf{v}, \mathbf{w} \in V에 대해, (\mathbf{u} + \mathbf{v}) + \mathbf{w} = \mathbf{u} + (\mathbf{v} + \mathbf{w}).
  4. 영벡터의 존재: 벡터 공간 V에는 영벡터 \mathbf{0}가 존재하여, 임의의 벡터 \mathbf{v} \in V에 대해 \mathbf{v} + \mathbf{0} = \mathbf{v}가 성립.
  5. 역벡터의 존재: 임의의 벡터 \mathbf{v} \in V에 대해, 벡터 -\mathbf{v} \in V가 존재하여 \mathbf{v} + (-\mathbf{v}) = \mathbf{0}가 성립.
  6. 스칼라 곱의 닫힘성: 임의의 스칼라 a와 벡터 \mathbf{v} \in V에 대해, a \mathbf{v} \in V.
  7. 스칼라 곱의 결합법칙: 임의의 스칼라 a, b와 벡터 \mathbf{v}에 대해, a(b \mathbf{v}) = (ab) \mathbf{v}.
  8. 단위원의 존재: 임의의 벡터 \mathbf{v} \in V에 대해, 1\mathbf{v} = \mathbf{v} (여기서 1은 스칼라).
  9. 스칼라 곱의 분배법칙 (스크래 곱과 벡터 덧셈의 결합): 임의의 스칼라 a와 벡터 \mathbf{u}, \mathbf{v} \in V에 대해, a(\mathbf{u} + \mathbf{v}) = a \mathbf{u} + a \mathbf{v}.
  10. 벡터 덧셈의 분배법칙 (벡터와 스칼라 덧셈의 결합): 임의의 스칼라 a, b와 벡터 \mathbf{v} \in V에 대해, (a + b) \mathbf{v} = a \mathbf{v} + b \mathbf{v}.

이 공리들을 모두 만족하는 집합 V를 벡터 공간이라고 한다.

기저와 차원

기저

벡터 공간 V의 기저(basis)는 V의 모든 원소(벡터)를 선형 결합으로 생성할 수 있는, 선형 독립인 벡터들의 집합이다. 일반적으로 벡터 공간 V의 벡터 집합 \{\mathbf{v}_1, \mathbf{v}_2, \cdots, \mathbf{v}_n\}가 V의 기저가 되는 조건은 다음과 같다:

  1. 선형 독립: 벡터들 \mathbf{v}_1, \mathbf{v}_2, \cdots, \mathbf{v}_n이 선형 독립이라는 것은, a_1 \mathbf{v}_1 + a_2 \mathbf{v}_2 + \cdots + a_n \mathbf{v}_n = \mathbf{0}일 때 모든 스칼라 a_i들이 0이어야 한다는 의미이다.

  2. 생성 집합: 벡터 공간 V의 모든 벡터 \mathbf{v} \in V는 기저 벡터들의 선형 결합으로 표현될 수 있다. 즉, 임의의 \mathbf{v} \in V에 대해, a_1 \mathbf{v}_1 + a_2 \mathbf{v}_2 + \cdots + a_n \mathbf{v}_n = \mathbf{v}가 성립하는 \{a_1, a_2, \cdots, a_n\}라는 스칼라들이 존재한다.

차원

벡터 공간 V의 차원(dimension)은 그 공간의 기저를 이루는 벡터의 수를 의미한다. 이를 \dim(V)로 표기한다. 예를 들어, \mathbb{R}^n 공간의 차원은 n이며, 이는 표준 기저 \{ \mathbf{e}_1, \mathbf{e}_2, \cdots, \mathbf{e}_n \}의 개수와 같다. 여기서 \mathbf{e}_i는 i번째 원소만 1이고 나머지 모든 원소는 0인 벡터이다.

예시

  • \mathbb{R}^2 벡터 공간: $$ \mathbb{R}^2 $$는 2차원 공간이며 표준 기저는 \{ [1, 0], [0, 1] \}로 이루어진다.
  • \mathbb{R}^3 벡터 공간: $$ \mathbb{R}^3 $$는 3차원 공간이며 표준 기저는 \{ [1, 0, 0], [0, 1, 0], [0, 0, 1] \}로 이루어진다.

선형 변환

선형 변환(linear transformation)은 한 벡터 공간에서 다른 벡터 공간으로 향하는 함수로, 다음 조건을 만족한다:

  1. 덧셈의 보존: T(\mathbf{u} + \mathbf{v}) = T(\mathbf{u}) + T(\mathbf{v}).
  2. 스칼라 곱의 보존: T(a \mathbf{u}) = a T(\mathbf{u}).

여기서 T는 선형 변환이며, \mathbf{u}와 \mathbf{v}는 동일한 벡터 공간에서의 벡터, a는 스칼라이다.

예시

  • 회전 변환: 2차원 공간에서의 회전 변환은 회전 행렬을 사용하여 벡터를 변환한다.
  • 확대/축소 변환: 벡터의 크기를 일정 배율로 확대하거나 축소하는 변환이다.

행렬과 선형 변환

행렬은 선형 변환을 나타낼 수 있는 도구이다. 행렬 A를 통해 선형 변환 T를 정의할 수 있으며, 다음과 같은 관계를 갖는다:

T(\mathbf{v}) = A \mathbf{v}

여기서 \mathbf{v}는 벡터이며, A는 적절한 크기의 행렬이다.
