벡터(vector)는 크기와 방향을 가지는 물리량으로, 평면 상에서의 벡터는 주로 2차원 좌표계에서 다루어진다. 벡터는 기하학적 해석을 통해 이해할 수 있으며, 이는 벡터의 기본 성질, 연산 및 응용을 포함한다.

벡터의 표현

평면에서의 벡터는 두 점 (x_1, y_1)와 (x_2, y_2)를 연결하는 직선으로 시각화할 수 있다. 이를 수학적으로 나타내면 다음과 같다.

\mathbf{v} = (v_1, v_2)

여기서 \mathbf{v}는 벡터로서, v_1과 v_2는 벡터의 성분(component)을 의미한다.

벡터의 덧셈과 뺄셈

두 벡터 \mathbf{a} = (a_1, a_2)와 \mathbf{b} = (b_1, b_2)의 덧셈은 각 성분별로 더하는 것이다:

\mathbf{a} + \mathbf{b} = (a_1 + b_1, a_2 + b_2)

뺄셈의 경우에도 동일하게 성분별로 뺄셈을 적용한다:

\mathbf{a} - \mathbf{b} = (a_1 - b_1, a_2 - b_2)

벡터의 스칼라 곱

벡터 \mathbf{v} = (v_1, v_2)와 스칼라 \lambda의 곱은 각 성분에 \lambda를 곱하는 것이다:

\lambda \mathbf{v} = (\lambda v_1, \lambda v_2)

벡터의 크기 (Norm)

벡터 \mathbf{v} = (v_1, v_2)의 크기 (또는 길이)는 다음과 같이 계산한다:

\|\mathbf{v}\| = \sqrt{v_1^2 + v_2^2}

단위 벡터

크기가 1인 벡터를 단위 벡터라고 한다. 벡터 \mathbf{v}의 단위 벡터는 다음과 같이 구할 수 있다:

\mathbf{\hat{v}} = \frac{\mathbf{v}}{\|\mathbf{v}\|}

내적 (Dot Product)

두 벡터 \mathbf{a} = (a_1, a_2)와 \mathbf{b} = (b_1, b_2)의 내적은 다음과 같다:

\mathbf{a} \cdot \mathbf{b} = a_1 b_1 + a_2 b_2

내적의 결과는 스칼라 값이다.

두 벡터 사이의 각도

두 벡터 \mathbf{a}와 \mathbf{b} 사이의 각도 \theta는 내적을 이용하여 구할 수 있다:

\cos \theta = \frac{\mathbf{a} \cdot \mathbf{b}}{\|\mathbf{a}\| \|\mathbf{b}\|}

벡터의 직교성과 평행성

직교성 (Orthogonality)

두 벡터 \mathbf{a}와 \mathbf{b}가 직교한다는 것은, 이 두 벡터 사이의 내적이 0이라는 의미이다:

\mathbf{a} \cdot \mathbf{b} = 0

이 경우, 두 벡터는 서로 수직이다.

평행성 (Parallelism)

두 벡터 \mathbf{a}와 \mathbf{b}가 평행하다는 것은, 한 벡터가 다른 벡터의 스칼라 배임을 의미한다. 즉, 어떤 스칼라 \lambda가 존재하여 다음이 성립한다:

\mathbf{a} = \lambda \mathbf{b}

벡터의 선형 결합

두 벡터 \mathbf{a}와 \mathbf{b}의 선형 결합은 두 벡터와 스칼라 \alpha, \beta를 사용하여 새로운 벡터를 만드는 것이다 :

\mathbf{c} = \alpha \mathbf{a} + \beta \mathbf{b}

기하학적 응용

벡터는 기하학적 문제를 해결하는 데 유용하게 사용된다. 여기서는 몇 가지 응용 예시를 소개한다.

벡터에 의한 점의 이동

벡터 \mathbf{v} = (v_1, v_2)를 사용하여 점 (x, y)를 이동시킬 수 있다. 새로운 점 (x', y')는 다음과 같이 계산된다:

(x', y') = (x + v_1, y + v_2)

두 점 사이의 거리

두 점 (x_1, y_1)와 (x_2, y_2) 사이의 거리 d는 두 점을 연결하는 벡터의 크기로 표현할 수 있다:

d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}

벡터를 이용한 삼각형의 면적

벡터 \mathbf{u} = (u_1, u_2)와 \mathbf{v} = (v_1, v_2)를 사용하여 삼각형의 면적을 계산할 수 있다. 삼각형의 면적 A는 두 벡터의 외적의 크기를 사용하여 다음과 같이 나타낼 수 있다 :

A = \frac{1}{2} |u_1 v_2 - u_2 v_1|

공간에서의 확장

벡터의 개념은 3차원 공간으로 확장될 수 있다. 이 경우 다음과 같은 추가 개념들이 있다:

3차원 벡터

3차원에서의 벡터는 다음과 같이 나타낼 수 있다:

\mathbf{v} = (v_1, v_2, v_3)

벡터의 크기

3차원 벡터의 크기는 다음과 같이 계산된다:

\|\mathbf{v}\| = \sqrt{v_1^2 + v_2^2 + v_3^2}

벡터의 내적

3차원 벡터 \mathbf{a} = (a_1, a_2, a_3)와 \mathbf{b} = (b_1, b_2, b_3)의 내적은:

\mathbf{a} \cdot \mathbf{b} = a_1 b_1 + a_2 b_2 + a_3 b_3

벡터의 외적 (Cross Product)

3차원 벡터의 외적은 다음과 같은 벡터로 구할 수 있다:

\mathbf{a} \times \mathbf{b} = (a_2 b_3 - a_3 b_2, a_3 b_1 - a_1 b_3, a_1 b_2 - a_2 b_1)

외적의 결과는 새로운 벡터로서, 두 벡터에 수직인 벡터이다.

두 벡터 사이의 각도

3차원에서도 두 벡터 \mathbf{a}와 \mathbf{b} 사이의 각도 \theta는 다음과 같이 구할 수 있다:

\cos \theta = \frac{\mathbf{a} \cdot \mathbf{b}}{\|\mathbf{a}\| \|\mathbf{b}\|}
