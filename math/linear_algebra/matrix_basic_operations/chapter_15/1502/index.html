양의 정부호 행렬의 정의

양의 정부호 행렬 \mathbf{A}은 모든 영 아닌 벡터 \mathbf{x} \in \mathbb{R}^n에 대해 다음을 만족한다:

\mathbf{x}^T \mathbf{A} \mathbf{x} > 0

행렬의 고윳값

양의 정부호 행렬의 고유한 성질 중 하나는 모든 고윳값이 양수라는 것이다. 즉, 행렬 \mathbf{A}의 모든 고윳값 \lambda_i에 대해,

\lambda_i > 0

체르노프 조건

체르노프 조건(Cholesky condition)은 양의 정부호 행렬이 상삼각행렬의 형태로 분해될 수 있음을 나타내며, 다음과 같은 분해가 가능한다:

\mathbf{A} = \mathbf{L} \mathbf{L}^T

여기서 \mathbf{L}은 하삼각 행렬이다.

제곱 형식

양의 정부호 행렬 \mathbf{A}와 관련된 모든 제곱형은 항상 양수이다. 즉, 모든 \mathbf{x} \neq 0에 대해서,

\mathbf{x}^T \mathbf{A} \mathbf{x} > 0

행렬식

양의 정부호 행렬의 행렬식은 양수이다. 즉,

\det(\mathbf{A}) > 0

주대각 성분

양의 정부호 행렬 \mathbf{A}의 모든 주대각 성분은 양수이다:

a_{ii} > 0 \quad \text{for all } i

주어진 부분 행렬

양의 정부호 행렬의 모든 대각선 부분 행렬(principal submatrix) 역시 양의 정부호이다. 이를 통해 하위 행렬에서도 해당 성질을 보장할 수 있다.

두 양의 정부호 행렬의 곱

두 양의 정부호 행렬 \mathbf{A}와 \mathbf{B}의 곱 \mathbf{C} = \mathbf{A} \mathbf{B}는 대체로 \mathbf{C}가 양의 정부호 행렬이 아님을 표시할 수 있다. 그러나 두 행렬의 경우 각각의 성분이 양수이어야 한다.

내적과의 관계

양의 정부호 행렬 \mathbf{A}는 공간을 변화시킬 때 벡터간의 내적 \mathbf{x}^T \mathbf{A} \mathbf{y}가 양의 준정부호 값을 가짐을 보장한다.

고윳값 분해

양의 정부호 행렬 \mathbf{A}는 항상 고윳값 분해(Eigenvalue Decomposition)를 통해 표현될 수 있으며, 이때 모든 고윳값들은 양수이다. 고윳값 분해는 다음과 같다:

\mathbf{A} = \mathbf{Q} \mathbf{\Lambda} \mathbf{Q}^T

여기서 \mathbf{Q}는 직교행렬이고, \mathbf{\Lambda}는 양의 고윳값을 갖는 대각행렬이다.

허미션 양의 정부호

복소수 행렬의 경우, 양의 정부호 행렬의 정의는 동일하지만 행렬 \mathbf{A}가 허미션(Hermitian) 행렬, 즉 \mathbf{A} = \mathbf{A}^H임을 추가로 요구한다. 여기서 \mathbf{A}^H는 \mathbf{A}의 켤레전치이다.

실용적인 응용

양의 정부호 행렬은 수많은 응용 분야에서 중요한 역할을 한다. 예를 들어, 최적화 문제에서 목표함수가 양의 정부호 행렬을 사용하는 경우, 해당 문제가 항상 전역 최적 해를 갖는다는 것을 보장한다.

양의 정부호 행렬은 선형대수학과 다차원 분석에서 중요한 개념이다. 이 행렬들은 중요한 수학적 성질을 갖고 있어 다양한 분야에서 유용하게 사용된다.
